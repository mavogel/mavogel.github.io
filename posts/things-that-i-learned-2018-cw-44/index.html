Things that I learned in week 44

November 4, 2018 One minute read
things-that-i-learned
npm • restify • inversify

npm libraries

The following libraries should be used with pinned versions - restify@6.3.2 - and inversify-restify-utils@3.3.1

Not even with a micro change restify@6.3.4. It is incompatible with the inversify utils. You get a Constructor already exists error.

Semantic versioning is not always reliable! I found a repository pointing out this error as well.
